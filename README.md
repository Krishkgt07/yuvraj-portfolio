## 🚀 Quick Start

### 1. Clone & Install

```bash
git clone https://github.com/YOUR_USERNAME/yuvraj-portfolio.git
cd yuvraj-portfolio
npm install
```

### 2. Run Development Server

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000)

### 3. Build for Production

```bash
npm run build
```

## ⚡ Performance

- ✅ Zero 3D libraries (no Three.js, Spline, WebGL)
- ✅ CSS transitions only — no heavy JS animation loops
- ✅ IntersectionObserver for scroll reveals
- ✅ Static ambient glows (no mouse tracking)
- ✅ Target: Lighthouse 90+
